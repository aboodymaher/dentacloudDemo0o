
# DentaCloud - نظام إدارة عيادة الأسنان 🦷

تطبيق متطور وسلس لإدارة عيادات الأسنان، مصمم لتوفير تجربة مستخدم استثنائية لكل من الطبيب وموظف الاستقبال.

## ✨ المميزات الرئيسية
- **لوحة تحكم ذكية**: عرض إحصائيات سريعة عن عدد المرضى، المواعيد، والدخل.
- **إدارة المرضى**: إضافة مرضى جدد، تتبع السجلات الطبية، والتواصل المباشر عبر WhatsApp.
- **مخطط الأسنان التفاعلي**: تسجيل الإجراءات على كل سن بدقة باستخدام نظام Palmer Notation.
- **نظام محاسبي متكامل**: تتبع التكاليف، الدفعات المقدمة، والمبالغ المتبقية لكل مريض.
- **جدول المواعيد**: عرض المواعيد بشكل تقويمي منظم.
- **تقارير تحليلية**: رسوم بيانية توضح أداء العيادة وتوزيع أنواع العلاج.
- **استشارة ذكية (AI)**: مدعوم بتقنية Gemini لتقديم نصائح طبية ذكية بناءً على التشخيص.

## 🚀 التقنيات المستخدمة
- **React 19**
- **Tailwind CSS** (للواجهات المتجاوبة)
- **Lucide React** (للأيقونات)
- **Recharts** (للرسوم البيانية)
- **Google Gemini API** (للذكاء الاصطناعي)

## 🛠️ كيفية التشغيل
1. قم بتحميل الملفات.
2. تأكد من وجود مفتاح API الخاص بـ Gemini في متغيرات البيئة باسم `API_KEY`.
3. افتح `index.html` في المتصفح (يدعم التطبيق العمل عبر ESM modules).
